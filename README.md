# fizzbuzzwoof
Fizzbuzz Woof 

Divisible by 3 => "fizz"
Divisible by 5 => "buzz"
Divisible by 3 and 5 => "fizzbuzz"
Divisible by 7 => "woof"

Number contains 3 => "fizz"
Number contains 5 => "buzz"
Number contains 7 => "woof" 

Fizzbuzz Woof 35 => Fizz Buzz Buzz Woof 
Fizzbuzz Woof 33 => fizz fizz fizz
Fizzbuzz Woof 77 => woof woof woof
